## 概述

这是仿照[redux官网的todo-demo](https://github.com/reduxjs/redux/tree/master/examples/todos，自己用react不用redux编写的todo。有2个特点：
1. 精简的es6语法。
2. 非常好的看出数据的流动和改动。

个人认为这本身就和redux就有点像了，待我学完redux再说。

## 安装

```
npm install
```

## 开始

```
npm start
```
